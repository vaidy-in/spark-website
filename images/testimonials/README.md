# Testimonial Images

This directory contains profile photos for testimonials on the marketing site.

## Required Images

### Chirag Doshi
- **Filename**: `chirag-doshi.jpg`
- **Recommended size**: 160x160px minimum (supports retina displays at 80px display size)
- **Format**: JPG or PNG
- **Shape**: Square (will be displayed as circular)
- **Quality**: Professional headshot, well-lit, clear background

## Image Guidelines

- Use square images (1:1 aspect ratio)
- Minimum 160x160px for retina display support
- Professional, high-quality photos
- Clear, well-lit headshots
- Simple or neutral backgrounds work best
- File size: aim for under 50KB per image
